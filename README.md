# Manan Portfolio (Next.js)

This portfolio now runs on Next.js with the App Router.

## Run locally

1. Install Node.js 20+.
2. Install dependencies:

```bash
npm install
```

3. Start dev server:

```bash
npm run dev
```

4. Open `http://localhost:3000`.

## Notes

- Main UI files are in `app/page.js` and `app/globals.css`.
- Legacy static files (`index.html`, `styles.css`) are retained but not used by Next.js.
